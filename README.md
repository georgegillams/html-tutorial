# HTML Tutorial

A quick-start app that uses a http-server to serve a single html file, which will automatically restart each time changes are made.

## Prerequisites


### Install Node.js

Install [Node.js](https://nodejs.org/en/) (The LTS version is probably best).

### Clone this repo

 1. Open Terminal.
 2. Paste in `git clone git@github.com:georgegillams/html-tutorial.git`, and hit "return".
 3. Enter the cloned folder by pasting in `cd html-tutorial` and hitting "return".
 4. In your terminal window, type in `npm ci` and hit "return".

## Starting the server

 1. Open Terminal.
 2. Enter the cloned folder by pasting in `cd html-tutorial` and hitting "return".
 3. Type `npm start` and hit "return".
 4. Your web browser should now be showing the site at `http://localhost:8080`.
